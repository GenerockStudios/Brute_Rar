   Ce package contient l'archiveur multifonctions WinRAR.

   Les caractéristiques de WinRAR sont les suivantes :

 * WinRAR introduit un algorithme de compression unique. Il permet 
   un meilleur taux de compression que d'autres utilitaires de compression 
   pour les fichiers exécutables, les bibliothèques d'objets, les fichiers 
   texte volumineux, etc.

 * Le format RAR ne limite pas la taille des archives et des fichiers, ni le
   nombre de fichiers dans une archive. Ces valeurs peuvent être limitées
   par les systèmes de fichiers ou les systèmes d'exploitation ainsi que
   par la mémoire disponible.

 * WinRAR offre une prise en charge complète des archives RAR et ZIP 2.0, 
   il décompresse également les archives CAB, ARJ, LZH, TAR, GZ, UUE, BZ2, 
   JAR, ISO, Z, 7Z, LZ, ZST.

 * WinRAR comporte deux interfaces, une classique sous Windows et une 
   pour le mode ligne de commande.

 * WinRAR offre la possibilité de créer des archives solides dont le 
   taux de compression est augmenté de 10% à 50% par rapport aux méthodes 
   traditionnelles, particulièrement dans le cas d'un grand nombre de 
   petits fichiers.

 * WinRAR offre la possibilité de créer des archives SFX personnalisables.

 * WinRAR offre la possibilité de créer des archives multi-volumes SFX.

 * L'enregistrement de restauration facultatif permet de protéger les archives
   contre les dommages.

 * WinRAR offre un grand nombre de fonctions: chiffrement des fichiers,
   ajout de commentaires d'archives et protection de la sécurité
   des fichiers NTFS.
